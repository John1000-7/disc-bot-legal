# Политика конфиденциальности

Бот собирает и хранит следующие данные:
- ID пользователя Discord
- ID сервера (guild ID)
- Роли, назначенные ботом
- Данные, связанные с функционалом турниров/бампов (при наличии)

Бот НЕ собирает:
- Личные данные (email, телефон, платёжную информацию)
- Содержимое личных сообщений

Данные хранятся в базе данных бота и используются исключительно для работы функционала (роли, турниры и т.д.). Данные удаляются при удалении бота с сервера или по запросу пользователя.

Контакт для вопросов/запроса удаления данных: [твой контакт/сервер поддержки]
